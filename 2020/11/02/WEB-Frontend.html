Les Langages

Donc comme nous avons vu juste avant. Après une requête, le serveur envois des information au client. Le client les interprète et les affiche ensuite sous forme de page Web. Pour les interpréter le clients utilisent des langages informatiques.

Ce sont les langages front end. Il y en a 3.

  1. HTML
  2. CSS
  3. Javascripts

Le navigateur Web lit ces langages pour les afficher de façon à ce que les humains le lise. C’est le site Web comme vous le voyez a la fin.

Voici quelques exemple de code écrit en HTML CSS Javascript. Et ce qu’ils rendent au final.

Eh bien sur cela vaut aussi pour Dhaki. Il est bien évidemment écrit en HTML CSS Javascript.

  • Le HTML est la pour écrire le contenu pure. Les image le texte, les titres ,les liens. Mais simplement ce qui est question de contenu non d'esthétique. Un peu comme le squelette d’un corp humain. C’est vraiment la structure de base.

  • Le CSS lui est la pour mettre des couleur faire la mise en page. Rendre le site agréable à lire. Pour continuer la métaphore ce serait la peau d’une personne, ses cheveux , sa coiffure, ses habits, la couleur de ses yeux etc…

  • Pour finir le Javascript s'occupe des actions un peu plus complexe comme les animations, transitions, ouverture de menu depuis des boutons etc… Ce serait dans cette métaphore les muscles du corp humain.

Ces trois langages sont donc les langages clients. Cela fait partie du développement Front end.
